# react-kit

> webpack: @2
>
> react: @15.6.1
>
> react-router-dom: @4.2.2

## Insall

```
git clone git@github.com:idsbllp/react-kit.git
```

## Usage

``` bash
rm -rf .git

# install dependencies
(c)npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

```
